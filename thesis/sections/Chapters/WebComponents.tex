\htwo{Web Components}

\hthree{Standard}

Web Components ermöglichen es "Client-Side"-Code wiederverwendbar zu machen. Dabei wurden drei neue Technologien eingeführt:

\begin{itemize}
    \item Custom Elements
    \item Shadow DOM
    \item HTML Templates
\end{itemize}
